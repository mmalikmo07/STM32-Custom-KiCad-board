# STM32 Custom PCB

A custom STM32-based development board designed from scratch in KiCad.
Programmable over USB with SWD debug access.

## Features
- STM32 microcontroller
- USB-C connectivity for power and programming
- SWD + UART debug headers
- Crystal oscillator (Pierce configuration)
- Switching regulator power supply
- BOOT0 toggle for bootloader / run mode switching
- Ferrite bead filtering on VDDA analog supply

## Tools Used
- KiCad 8
- STM32CubeMX
- STM32CubeIDE
